# Guessing-Game
Number Guessing Game

A simple JavaScript Number Guessing Game where the computer generates a random number, and the player tries to guess it using hints.

🚀 Features:
Generate a random number within a user-defined range.
Give hints if the guess is too high or too low.
Keep asking until the correct number is guessed.
Option to quit the game anytime by typing quit.

🛠️ Technologies Used:
HTML
JavaScript
